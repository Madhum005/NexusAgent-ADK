# NexusAgent-ADK

A Single Agent Multi-Tool AI Assistant built using Google Agent Development Kit (ADK) and Gemini. 
This project demonstrates how a single intelligent agent can seamlessly interact with multiple tools to perform different tasks through natural language conversations.

## Overview

NexusAgent-ADK combines tool-calling capabilities with the power of large language models, enabling users to access multiple functionalities from a single interface.
The agent intelligently determines which tool to use based on user requests and returns accurate responses.

## Features

* Single Agent Architecture
* Multi-Tool Integration
* Current Time Retrieval
* Weather Information Lookup
* Natural Language Interaction
* Gemini-Powered Reasoning
* Easy Deployment with ADK

## Technologies Used

* Python
* Google Agent Development Kit (ADK)
* Gemini 2.5 Flash
* Pyngrok
* ZoneInfo Library

## Example Queries

* What is the current time in Tokyo?
* Show the weather in London.
* Tell me the time in Chennai.
* What is the weather like in New York?

## Installation

```bash
pip install google-adk
pip install pyngrok
```

## Running the Project

```bash
adk web
```

or

```bash
adk run nexus_agent
```
